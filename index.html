Install:

Add this sources.list.d entry (DEB822-style):

Types: deb
URIs: https://xlibre-debian.github.io/[system]
Suites: main
Components: stable
Signed-By: /etc/apt/keyrings/NexusSfan.asc


Where [system] can be debian or devuan depending if you have systemd or not.

Now, run wget -O - https://mrchicken.nexussfan.cz/publickey.asc | sudo tee /etc/apt/keyrings/NexusSfan.asc
This will set up the publickey.

You can install XLibre with sudo apt install xlibre xlibre-archive-keyring
