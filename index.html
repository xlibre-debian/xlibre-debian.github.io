Install:

Add this sources.list.d entry (DEB822-style):

Types: deb
URIs: https://xlibre-debian.github.io/[system]/
Suites: main
Components: stable
Architectures: amd64
Signed-By: /usr/share/keyrings/NexusSfan.pgp


Where [system] can be debian or devuan depending if you have systemd or not.

Now, run curl -fsSL https://mrchicken.nexussfan.cz/publickey.asc | gpg --dearmor | sudo tee /usr/share/keyrings/NexusSfan.pgp > /dev/null
This will set up the publickey.

After updating the package cache, you can install XLibre with sudo apt install xlibre xlibre-archive-keyring

If you are using Nvidia's proprietary driver on Debian based distros (but not Ubuntu) then install xserver-xlibre-nvidia-conf

----------

Uninstall:


sudo apt remove xlibre xserver-xlibre*
sudo apt install xorg
sudo rm /etc/apt/sources.list.d/xlibre-debian.sources /usr/share/keyrings/NexusSfan.pgp
